Contentful integration with node.

## Install dependencies

```bash
yarn # npm install
```

## Start development server

```bash
yarn dev # npm run dev
```

## Code linting

```bash
yarn lint # npm run lint
```

## Run tests

```bash
yarn test # npm run test
```

## Build for production

```bash
yarn build # npm run build
```
